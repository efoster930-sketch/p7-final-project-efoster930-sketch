# Topics in Computer Science 2026
Final Project for Topics in CS 2026


**Overview**

The Final Project is a culminating experience in which you will design, plan, and develop a program based on one of the major topics studied this year: Data Science, Game Design, Artificial Intelligence, or Cybersecurity. You will propose your idea, receive feedback, and then execute your project through structured planning, development, and reflection.

**Objectives:**
 * Apply concepts learned throughout the course to a self-directed project
 * Demonstrate problem-solving, planning, and technical development skills
 * Create a functional and meaningful program
 * Communicate your design process and decisions clearly

**Project Requirements:**
 * Your project must be based on one of the following areas: Data Science, Game Design, Artificial Intelligence, or Cybersecurity
 * Your project must result in a working program or application
 * Your program should demonstrate appropriate complexity for a full project (not a minimal or trivial program)
 * You must follow a structured development process including proposal, planning, implementation, and reflection

**Project Phases**

**Phase 1: Proposal**

You will submit a proposal outlining your project idea for approval.

Proposal Must Include
 * Project Title
 * Selected Topic Area (Data, Game Design, AI, or Cybersecurity)
 * Description of the Program
 * Target User or Audience
 * Key Features and Functionality
 * Tools/Technologies You Plan to Use
 * Initial Timeline or Plan

**Phase 2: Planning**

After approval, you will create a detailed plan for your project.

Planning Must Include
 * Breakdown of features into manageable tasks
 * Timeline with checkpoints or milestones
 * Basic design documentation (flowcharts, sketches, or pseudocode)
 * Identification of potential challenges and solutions

**Phase 3: Development**

You will build your program according to your plan.

Expectations:
 * Consistent progress during class time
 * Regular testing and debugging
 * Iterative improvements based on feedback
 * Proper use of version control (GitHub repository)

**Phase 4: Final Submission and Reflection**

You will submit your completed project along with documentation.

Final Submission Must Include:
 * Completed Program
 * Well-documented code with comments
 * README file explaining your project
 * Reflection on your process and learning

**Grading Breakdown (200 Points)**

Proposal (25 Points):
 * Clear and complete idea (10)
 * Appropriate scope and complexity (10)
 * Organization and clarity (5)

Planning (35 Points):
 * Task breakdown and timeline (15)
 * Design documentation (10)
 * Identification of challenges (10)

Development Process (60 Points):
 * Effective use of class time (20)
 * Progress and iteration (20)
 * Problem-solving and debugging (20)

Final Product (60 Points):
 * Functionality and completeness (25)
 * Complexity and creativity (20)
 * Code quality and organization (15)

Reflection and Documentation (20 Points):
 * Quality of README (10)
 * Thoughtful reflection on learning and challenges (10)

Expectations:
 * Projects must be your own work
 * Collaboration is allowed for brainstorming and feedback, but not for completing the project
 * AI tools may be used for support, but not to generate your entire project
 * Stay on task and use class time productively

Suggested Project Ideas by Topic

Data Science
 * Data visualization dashboard
 * Analysis of a real-world dataset
 * Prediction model using basic statistics

Game Design
 * 2D game using a graphical library
 * Puzzle or strategy game with scoring system
 * Game with increasing difficulty or levels

Artificial Intelligence
 * Simple chatbot
 * Rule-based decision system
 * Basic machine learning model

Cybersecurity: 
 * Password generator and strength checker
 * Secure login system
 * Encryption/decryption tool

Timeline:
 * Proposal Submission: 5/4/2026
 * Planning Checkpoint: 5/7/2026
 * Development Checkpoints: Ongoing
 * Final Submission: 6/2/2026


**Final Note**

This project is your opportunity to build something meaningful and demonstrate everything you have learned this year. Choose a topic that interests you and challenge yourself to create something you are proud of.
